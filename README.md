# Portfolio - Ursula Ngalubenge Nandoy

Ce projet est un portfolio moderne, responsive et bilingue (FR/EN) conçu pour présenter mes compétences en développement web, mes projets et mes réseaux sociaux.

## Technologies Utilisées

*   **HTML5** : Structure sémantique du site.
*   **Tailwind CSS (CDN)** : Framework CSS utilitaire pour un design rapide et personnalisé.
*   **DaisyUI** : Plugin de composants pour Tailwind CSS (utilisé pour les thèmes, les boutons et les cartes).
*   **JavaScript (Vanilla)** : Gestion du changement de langue, du mode sombre/clair et de l'interactivité du menu mobile.
*   **Font Awesome** : Bibliothèque d'icônes pour les réseaux sociaux et l'interface.
*   **Google Fonts** : Polices "Inter" (sans-serif) et "JetBrains Mono" (code).

## Installation et Utilisation

Comme le projet utilise des CDN pour Tailwind et DaisyUI, aucune installation de dépendances (npm) n'est requise pour le moment.


## Fonctionnalités

*   **Mode Sombre/Clair** : Bascule dynamique grâce au système de thèmes de DaisyUI.
*   **Support Bilingue** : Système de traduction simple via JavaScript.
*   **Responsive Design** : Adapté aux mobiles, tablettes et ordinateurs.
*   **Menu Burger Mobile** : Navigation optimisée pour les petits écrans.
*   **Section Projets** : Cartes interactives avec liens vers les démos et le code source.
